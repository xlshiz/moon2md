# .mrexpt 文件格式说明

> 通过逆向工程分析 Moon+ Reader（静读天下）v8.x 版本的导出文件所得。

## 概述

`.mrexpt` 是 Moon+ Reader 导出的高亮与批注文件，为纯文本格式，UTF-8 编码。

## 文件结构

```
<文件头>        ← 3 行元数据
#               ← 分隔符
<记录 1>        ← 16 行字段
#
<记录 2>
#
...
#
<记录 N>
```

## 文件头

文件前 3 行：

```
0               ← 第 1 行: 格式版本号（始终为 "0"）
indent:false    ← 第 2 行: 缩进开关
trim:false      ← 第 3 行: 修剪开关
```

## 记录格式

每条记录固定 16 行，以 `#` 分隔：

### 字段对照表

```
行号  索引  示例值                         说明
──────────────────────────────────────────────────
1     0     26                             序号（递增）
2     1     数学女孩3：哥德尔不完备定理     书名
3     2     /sdcard/Books/xxx.epub         文件路径（原始）
4     3     /sdcard/books/xxx.epub         文件路径（小写）
5     4     52                             章节号/章节标识
6     5     0                              未知（通常为 0）
7     6     483                            章节内字符偏移位置
8     7     26                             高亮文本长度
9     8     -28160                         颜色编码（详见下文）
10    9     1671754358620                  时间戳（毫秒）
11    10    （空行）                        未知标志
12    11    （空行或批注内容）              批注/笔记
13    12    集合的外延表示法中...           高亮原文
14    13    1                              未知标志
15    14    0                              未知标志
16    15    0                              未知标志
```

### 时间戳

毫秒级 Unix 时间戳。示例：

```
1671754358620 → 2022-12-23 08:12:38
```

转换公式（Python）：

```python
from datetime import datetime
dt = datetime.fromtimestamp(1671754358620 / 1000)
# → 2022-12-23 08:12:38
```

### 颜色编码

Moon+ Reader 使用**32 位颜色补码**。颜色值存储为带符号整数，转换方式：

```python
def decode_color(code):
    """将颜色编码转为 6 位十六进制 RGB"""
    return hex(0xFFFFFFFF + int(code) + 1)[4:]
```

常见颜色对照：

| 编码 | 十六进制 | 颜色 |
|------|----------|------|
| `-65536` | `FF0000` | 红色 |
| `-16711936` | `00FF00` | 绿色 |
| `-16776961` | `0000FF` | 蓝色 |
| `-256` | `FFFF00` | 黄色 |
| `-28160` | `FF9200` | 橙色（默认） |
| `-65281` | `FF00FF` | 紫色 |
| `-16711681` | `00FFFF` | 青色 |

### 换行符

高亮原文中的 `<BR>` 标签表示换行。静读天下在导出 epub 中的多行内容时使用该标签而非 `\n`。

### 特殊符号

原文中的 `￼`（U+FFFC, OBJECT REPLACEMENT CHARACTER）出现在 epub 中包含图片、公式、注音等特殊元素的位置。这是静读天下导出时无法渲染的内容产生的替换符，原文件本身不包含这些字符。

## 与 .mrpro 的区别

| 特征 | .mrexpt | .mrpro |
|------|---------|--------|
| 用途 | 高亮/批注导出 | 完整备份（含书籍元数据、配置） |
| 格式 | 纯文本 | 二进制 / ZIP |
| 可读性 | 可直接查看 | 需专用工具 |
| 包含内容 | 仅高亮和批注 | 阅读进度、设置、书签等 |
| 文件大小 | 几 KB | 较大 |

## 格式验证

可使用以下方式验证 mrexpt 文件完整性：

```bash
# 统计高亮条数
grep -c '^#$' 文件.mrexpt

# 检查首行元数据
head -3 文件.mrexpt
# 应输出: 0 / indent:false / trim:false

# 查看颜色分布
grep -E '^-[0-9]+$' 文件.mrexpt | sort | uniq -c | sort -rn
```
